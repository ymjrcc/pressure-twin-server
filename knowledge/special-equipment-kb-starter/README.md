# 特种设备知识库问答种子包

这套资料用于快速验证 Markdown → 切片 → Embedding → 向量检索 → 重排 → 大模型回答的完整 RAG 链路。内容重点覆盖特种设备使用管理、固定式压力容器巡检、安全附件检查和异常处置。

## 使用边界

本知识库是根据公开法规、安全技术规范和政府解读材料整理的检索型摘要，不替代法规、TSG、国家标准原文，也不能作为设计、制造、检验或者行政执法的唯一依据。

当问题涉及以下情形时，回答应提示人工复核：

- 是否必须停用、报废、处罚或者构成重大事故隐患；
- 安全阀整定压力、容器允许工作参数、检验周期等具体技术参数；
- 涉及易燃、易爆、有毒、强腐蚀、高温、高压介质；
- 现场状态与设计文件、铭牌、检验报告不一致；
- 法规或安全技术规范版本可能已经更新。

## 推荐加载方式

递归加载本目录中的 `.md` 文件。切片时建议保留 YAML 元数据、标题层级和 `依据` 段落。

初始切片参数可以使用：

- 每个切片约 500～900 个中文字符；
- 重叠约 80～150 个字符；
- 优先按二级、三级标题切分；
- `source_url`、`document_no`、`status`、`equipment_types` 作为向量记录的 metadata；
- 检索结果返回前端时显示文件名、章节标题、依据文件和来源链接。

## 目录说明

- `01_法规与规范`：法规和安全技术规范的结构化摘要；
- `02_压力容器巡检`：可直接用于巡检问答的业务知识；
- `03_异常处置`：异常识别和初步处置边界；
- `04_问答辅助`：回答规则、常见问题和测试集。

## 版本说明

截至 2026-06-17，市场监管总局安全技术规范栏目仍列示 TSG 21—2016 第1号修改单；2025年公开的《固定式压力容器安全技术规程（修订稿）》属于征求意见稿，本知识库未将其作为现行依据。TSG 31—2025 已于 2026-01-01 起施行。
